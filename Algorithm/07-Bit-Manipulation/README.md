# Bit Manipulation

Operate directly on binary representations using AND, OR, XOR, shifts, etc.

## Sub-techniques
| Name | Description |
|------|-------------|
| `XOR` | XOR properties: x^x=0, x^0=x |
| `BitCounting` | Count set bits to identify duplicates |
| `BitManipulation` | General bit tricks (Gray code, ripple carry, …) |
