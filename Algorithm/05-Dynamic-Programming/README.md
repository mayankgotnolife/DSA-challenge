# Dynamic Programming

Break the problem into overlapping sub-problems, store results to avoid recomputation.

## Sub-techniques
| Name | Description |
|------|-------------|
| `Kadane` | Maximum/minimum subarray sum in O(n) |
| `DP` | Classic top-down / bottom-up DP |

## Problems

| 1443 | Minimum Distance to Type a Word Using Two Fingers | DP |
| 2554 | Minimum Total Distance Traveled | DP |
