# Binary Search

Binary search divides the search space in half each iteration, achieving O(log n).

## Sub-techniques
| Name | When to use |
|------|-------------|
| `BinarySearch` | Classic search on a sorted array |
| `ModifiedBinarySearch` | Rotated / irregular sorted arrays |
| `BinarySearchOnAnswer` | "Find the minimum X such that …" problems |
