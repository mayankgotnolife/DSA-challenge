# DSA Challenge

This repository tracks my daily progress in solving 5 Data Structures and Algorithms (DSA) problems in Java. Each day, I solve problems from platforms like LeetCode or HackerRank and commit the solutions here.

## Structure
- Each day has its own folder (e.g., `Day1`, `Day2`, etc.).
- Inside each day's folder, there are 5 Java files corresponding to the problems solved that day (e.g., `Problem1.java`, `Problem2.java`, etc.).
- Each Java file contains the solution code, along with comments explaining the approach.

## Goals
- Solve 5 problems per day.
- Commit all code daily to maintain a consistent streak.
- Improve problem-solving skills and algorithmic thinking.

## Platforms
- [LeetCode](https://leetcode.com/)
- [HackerRank](https://www.hackerrank.com/)

## How to Run
1. Clone the repository.
2. Navigate to the specific day's folder.
3. Compile and run the Java files using `javac ProblemX.java` and `java ProblemX`.
