# DSA Challenge 🚀

A curated collection of LeetCode solutions written in **Java**, organised into two
top-level categories:

```
DSA-challenge/
├── Data-Structure/    ← problems grouped by the container they use
└── Algorithm/         ← problems grouped by the technique they use
```

## Folder convention

Every problem folder contains **one sub-folder per algorithm approach** used.

```
Algorithm/01-Binary-Search/
└── 33-search-in-rotated-sorted-array/
    ├── README.md
    └── ModifiedBinarySearch/
        └── search-in-rotated-sorted-array.java

Data-Structure/07-Graph/
└── 1507-check-if-there-is-a-valid-path-in-a-grid/
    ├── README.md
    └── BFS/
        └── check-if-there-is-a-valid-path-in-a-grid.java
```

Problems that admit 2–3 well-known approaches each get their own sub-folder.
